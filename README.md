## Overview

Coding Conference Ticket is an event ticketing application designed to help users browse, book, and manage tickets for coding conferences efficiently. It provides a seamless user experience for attendees and organizers.

## Features

- Browse upcoming coding conferences
- Book tickets easily
- Manage ticket information
- Secure payment integrationFully responsive design
- Styled with TailwindCSS
- Built with modular React components

## Tech Stack

Frontend:

- React
- JavaScript (ES6+)
- TailwindCSS

Other Tools:

- Git
- React Router
- Vite

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/pro-mcode/demo-coding-conference.git
cd weather-now
npm install
```

## Usage

```bash
npm run dev
npm run build
npm run preview
```
